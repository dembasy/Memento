La ligne de commande

Gestion de documents

  • mkdir

    • J'utilise la commande mkdir pour créer un dossier

  • ls

    • La commande ls liste les fichiers et les sous-dossiers. (ls = list)

  • ls -l

    • Pour obtenir plus d'information, utiliser l'option -l (pour version longue)

  • ls -A

    • Pour afficher aussi les fichiers "cachés" (en fait ceux qui commencent par un point) utilisez l'option -A (pour All)

  • ls -Al

    • On peut aussi bien sûr mélanger les deux options

  • pwd

    • La commande pwd vous permet de savoir dans quel répertoire de l'arborescence vous êtes localisé. (pwd = present working directory)

  • mv

    • La commande mv sert à déplacer un dossier (ou fichier) à un endroit différent ou à le renommer.

  • cp

    • La commande cp fait une copie d'un fichier ou répertoire. (cp = copy)

  • rm (remove)

    • Elle permet de supprimer un fichier ou un répertoire. Attention, cette commande est irréversible

  • supo -s (super utilisateur)

    • c'est l'utilisateur qui a tout les droits sur l'ordinateur, il peut manipuler librement les éléments, certains indices nous permette de le remarquer comme un "#" et un root devant la session

  • apt update

    • Cette commande sert a mettre à jour la liste des paquets

  • apt upgrade

    • Cette commande sert à mettre à niveau

  • apt search

    • Sert à chercher et trouver un fichier

  • apt remove

    • c'est pour supprimer un logiciel

  • dpkg -i ...

    • sert à installer un logiciel en passant par un dépôt comme le ferais apt

  • less

    • permet d'afficher le continu'''

Repertoire

La commande cd vous permet de changer de répertoire (cd = change directory).
Quand vous ouvrez un terminal en mode utilisateur vous êtes dans votre répertoire local (/home/utilisateur). cd /home/utilisateur/dossier Le symbole "~" peut remplacer le chemin absolu vers votre répertoire personnel soit "/home/utilisateur/". cd ~/Desktop

La commande pwd vous permet de savoir dans quel répertoire de l'arborescence vous êtes localisé. (pwd = present working directory) La commande mv sert à déplacer un dossier (ou fichier) à un endroit différent ou à le renommer. (mv = move) La commande cp fait une copie d'un fichier ou répertoire.
